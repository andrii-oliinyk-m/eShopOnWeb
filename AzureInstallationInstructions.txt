========================================================
Step 1.
========================================================
Goal: 
	Install web apps 
		for the Web project (two instances), 
		for Public Api project,
	Install Azure SQL server and Azure SQL Database
--------------------------------------------------------
Actions:
	Run AzureInstallation.azcli in the Cloud Shell.
	Go to Azure Sql Database and set the firewall rule for the client IP.
========================================================
Step 2.
========================================================
Goal:
	Deploy the database schema to the Azure SQL Database.
--------------------------------------------------------
Actions:
	Set ConnectionStrings:CatalogConnection in appsettings.Development.json in the Web project.
	Set ConnectionStrings:IdentityConnection in appsettings.Development.json in the Web project.
	Run locally ./Apply-Migrations.bat.
========================================================
Step 3.
========================================================
Goal:
	Setup a dependency from Blazor to Public Api.
--------------------------------------------------------
Actions:
	Set baseUrls:apiBase in appsettings.Development.json in the Blazor project.
		--> baseUrls:apiBase must be set to the url of the web api application.
========================================================
Step 3.
========================================================
Goal:
	Publish Web Instance #1, Instance#2, PublicApi
--------------------------------------------------------
Actions:
	Publish Web Instance #1.
	Publish Web Instance #2.
	Publish PublicApi
